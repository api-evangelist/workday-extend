Workday has launched Workday Build, a comprehensive developer platform enabling organizations to create AI-powered solutions integrated with the Workday ecosystem. The platform features a developer copilot, unified development tools, and integrates Workday Flowise Agent Builder for creating custom AI agents. Through the Built on Workday program and marketplace, developers can distribute solutions to customers while building a growing global developer community.
